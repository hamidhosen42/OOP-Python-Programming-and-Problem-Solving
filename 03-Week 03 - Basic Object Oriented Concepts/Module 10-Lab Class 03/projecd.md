# Project
- Camera Application

#!======================
```
This Camera application is a desktop-based tool created using PyQt5 and OpenCV. The app allows users to capture images and record videos from their webcam. It features a user-friendly interface with capture and record buttons. The application also has functionalities to save the captured images and recorded videos with a timestamp, ensuring each file is uniquely named based on the date and time of capture.
```
